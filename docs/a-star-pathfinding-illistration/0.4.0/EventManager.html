    EventManager

    new EventManager()

    EventManager is not a real class, rather an interface which is applied with Object.makeEventManager() on an already existing Object.

    Please see the documentation of makeEventManager for more infomration of the usage.

    Members

    events :Array.<string>

    Events contains the defined events, if non where defined the array is empty.

    The frozen (== inmuteable) array Events is meant to represent the available events, which includes an runtime check on register, wether or not this event is defined

    Events has two special members

    • toString(): is a function which returns a string which is this array in a json format
    • contains(): is a function used by add, remove and dispatch to check if the event is allowed, returns true if the array is empty
